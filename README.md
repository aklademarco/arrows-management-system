# Church-management-system
An open-source platform for managing church workers, attendance, departments, events, and engagement through secure location-based check-ins.


## ✨ Features

- 📍 Geofence-based attendance
- 👥 Member management
- 🏢 Department management
- 📅 Event scheduling
- 🏆 Individual & department leaderboards
- 📊 Attendance analytics
- 🔐 Role-based access control
- 📈 Reports & exports

## 🛠 Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- NestJS
- PostgreSQL
- Drizzle ORM
- JWT Authentication
